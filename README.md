# FishEye

Site web de découverte de photographes indépendants, entièrement codé from scratch en HTML/CSS/JavaScript.

## Description

FishEye est une plateforme qui met en avant le talent de photographes indépendants du monde entier. Le site permet de découvrir les portfolios des photographes, de consulter leurs œuvres (photos et vidéos), de les contacter et d'interagir avec le contenu (likes, tri, lightbox).

## Comment le démarrer

Ce projet est un site statique HTML/CSS/JavaScript. Pour le lancer, il est recommandé d'utiliser un service comme l'extension Live Server dans VSCode :

1. Installez l'extension **Live Server** dans VSCode si ce n'est pas déjà fait
2. Ouvrez le dossier du projet dans VSCode
3. Faites un clic droit sur [`index.html`](index.html) et sélectionnez **"Open with Live Server"**
4. Cela lancera un serveur local (généralement sur http://127.0.0.1:5500)
